# Splunk Mastery Lab - Project Overview

## Purpose

This project demonstrates practical Splunk capability across data onboarding, SPL analytics, dashboards, detection engineering, and automated SOC triage.

It is designed as a portfolio lab for security analyst, SOC analyst, detection engineer, and Splunk-focused roles.

## What Recruiters and Reviewers Should Notice

- Built a custom Splunk app with dedicated indexes, sourcetypes, lookups, dashboards, macros, and saved searches.
- Created realistic security telemetry across identity, web/API, endpoint, cloud, and network domains.
- Built dashboards for SOC overview, identity threats, web/API security, endpoint detection, and cloud security.
- Wrote reusable SPL detections for brute force, impossible travel, suspicious PowerShell, suspicious egress, CloudTrail privilege escalation, and web attack behavior.
- Added an Agentic SOC Analyst that queries Splunk, runs detections, ranks findings, explains evidence, and generates visual reports.
- Included optional home-network telemetry workflows using Windows connection metadata, pktmon packet capture, and router syslog.

## Core Skills Demonstrated
